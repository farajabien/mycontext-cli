# MyContext AI CLI: The Context Engine for AI-Native Development

AI code generation often fails because it lacks deep, structured project awareness. **MyContext** solves this by generating a 100% accurate 'Living Brain' scaffold—PRDs, Type systems, and Component Architectures—that you can hand off to any AI agent to build the application with perfect alignment.

## 🎯 Project Overview
MyContext is not just a generator; it is a **Context Scaffold Engine**. It analyzes your codebase (or your simple description) and builds a high-fidelity "Brain" (`.mycontext/context.json`) that maps out every entity, user story, and component your app needs. This brain acts as the source of truth for your next AI agent (Antigravity, Cursor, etc.) to take over and build with zero hallucinations.

## ✨ Key Features
- **Brain Reconstruction (FSR)**: Scans any existing project and reconstructs a high-fidelity Living Brain from source code alone.
- **Agent Hand-off Scaffold**: Generates the exact PRD, types.ts, and component-list.json needed to "scope" another AI agent's work.
- **Living Brain Management**: Core functionality to initialize, analyze, and manage the project's 'Living Brain' (context.json).
- **Monorepo Awareness**: Automatically detects and synchronizes brains across complex multi-app monorepos.
- **Sync & Align**: Keeps your documentation and brain perfectly in sync with the actual evolution of your project.

## 🛠️ Technical Stack

- **Backend**: Node.js, TypeScript
