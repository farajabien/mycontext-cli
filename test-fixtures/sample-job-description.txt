Senior Full-Stack Engineer

Company: TechCorp Innovations
Location: Remote (US-based)
Type: Full-time

About the Role:
We are seeking an experienced Senior Full-Stack Engineer to join our growing engineering team. You will be responsible for designing, developing, and maintaining scalable web applications that serve millions of users worldwide.

Key Responsibilities:
- Design and implement robust, scalable web applications using modern tech stack
- Lead technical architecture decisions and mentor junior developers
- Collaborate with product managers and designers to deliver exceptional user experiences
- Write clean, maintainable, and well-tested code
- Optimize application performance and ensure security best practices
- Participate in code reviews and contribute to engineering best practices
- Drive technical innovation and continuous improvement initiatives

Required Qualifications:
- 5+ years of professional software engineering experience
- Expert-level proficiency in React, Next.js 14+, and TypeScript
- Strong experience with Node.js and Express/Fastify for backend development
- Proficiency with modern state management (Redux, Zustand, or similar)
- Experience with PostgreSQL or MongoDB databases
- Solid understanding of RESTful APIs and GraphQL
- Experience with cloud platforms (AWS, Vercel, or similar)
- Strong problem-solving skills and attention to detail
- Excellent communication and collaboration abilities
- Experience with Git, CI/CD pipelines, and modern development workflows

Nice to Have:
- Experience with system design and distributed systems
- Contributions to open source projects
- Technical blog writing or conference speaking
- Experience with microservices architecture
- Knowledge of Docker and Kubernetes
- Familiarity with test-driven development (TDD)
- Experience with design systems and component libraries
