# Test Project

A simple test application for testing the design pipeline resume functionality.

## Features
- User authentication
- Dashboard
- Settings

## Technical Requirements
- Next.js 14
- TypeScript
- Tailwind CSS


## Game Mechanics
- **Game Type**: Board game

- **Core Rules**: a 100 piece board where each user has 25 and 50 are emptties and take turns to reveal cells and whover reveals all of ops wins
