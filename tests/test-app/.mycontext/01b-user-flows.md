# E-Commerce Store User Flow Documentation

## 1. Primary User Journeys

### Journey 1: Browse & Purchase Flow
**Description:** Standard path for discovering and purchasing products

```text
[Home] → [Browse Catalog] → [Product Details] → [Add to Cart] → [Checkout] → [Order Confirmation]
```

**Steps:**
1. User lands on homepage
2. Clicks "Electronics" category
3. Filters by price range $100-$500
4. Selects product card
5. Chooses quantity (3) and clicks "Add to Cart"
6. Clicks cart icon and selects "Checkout"
7. Completes guest checkout form
8. Submits payment
9. Receives order confirmation

**Decision Points:**
- Guest vs logged-in checkout
- Payment method selection
- Shipping option choice

**Success Criteria:**
- Order appears in confirmation page
- Cart empties after purchase
- Order visible in account history (if logged in)
