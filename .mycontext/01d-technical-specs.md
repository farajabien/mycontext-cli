# Technical Specifications

## Architecture
The project is a monorepo managed by pnpm workspaces. The primary application is a command-line interface (CLI) tool located in `apps/cli`. This CLI is built with TypeScript and Node.js, using `commander.js` for command parsing. The core logic is architected around a system of specialized AI 'Sub-Agents', managed by a central `SubAgentOrchestrator`. Each agent is a class responsible for a specific domain (e.g., code generation, security, QA). Agents communicate and share state through a central 'Living Brain' (a structured JSON file), following a blackboard architectural pattern. The monorepo also contains several experimental Next.js applications used for testing, demonstration, and as targets for the CLI's capabilities.

## Technology Stack
- **Frontend:** Next.js, React, TypeScript, Tailwind CSS, shadcn/ui
- **Backend:** Node.js, TypeScript
- **Database:** Local storage
