# Technical Specifications

## Architecture
Monorepo structure with distinct directories for CLI, web interface, VS Code extension, and shared core functionalities.

## Technology Stack
- **Frontend:** React, HTML, CSS
- **Backend:** Node.js
- **Database:** JSON (Living DB), Option for external database integration

## API Endpoints
- **POST /context/generate**: Generates a context.json file based on input parameters. (Auth: Yes)
- **POST /code/evolve**: Triggers autonomous code evolution based on the context.json file. (Auth: Yes)

## Database Schema

### Table: contexts
| Column | Type | Constraints |
|---|---|---|
| id | UUID | PRIMARY KEY |
| name | VARCHAR(255) | NOT NULL |
| data | JSON | NOT NULL |
| created_at | TIMESTAMP | NOT NULL |
| updated_at | TIMESTAMP | NOT NULL |


### Table: users
| Column | Type | Constraints |
|---|---|---|
| id | UUID | PRIMARY KEY |
| username | VARCHAR(255) | NOT NULL |
| password_hash | VARCHAR(255) | NOT NULL |
| email | VARCHAR(255) | NOT NULL |
| created_at | TIMESTAMP | NOT NULL |
