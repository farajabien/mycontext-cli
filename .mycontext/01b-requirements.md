## Goals
- To provide a CLI tool that can initialize, analyze, and manage a 'Living Brain' (a comprehensive JSON context file) for any web project.
- To orchestrate a suite of specialized AI agents that can read from and write to the Living Brain to perform tasks like code generation, documentation, visual testing, and refactoring.
- To enable a spec-driven development workflow where changes to the spec (the Living Brain) can be used to automatically generate or update code, tests, and documentation.
